# ⚽ Football Squad Builder

An interactive web application that allows users to visually build and manage a football (soccer) team lineup using a drag-and-drop interface.

## 🚀 Features
- Drag-and-drop players onto a football pitch
- Supports basic formation layout
- Real-time visual feedback while placing players
- Persistent squad data using browser Local Storage
- Responsive design for desktop and mobile screens
- Clean and intuitive user interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Drag & Drop API
- Browser Local Storage

## 📌 Use Case
This project demonstrates front-end development skills, UI/UX thinking, and interactive state management without relying on external frameworks. It is suitable for learning, prototyping, and showcasing drag-and-drop functionality in web applications.
